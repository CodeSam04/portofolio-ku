# 🌐 Simple Portfolio Website

A lightweight, responsive, and modern **single-page portfolio website** built with **HTML5** and **Tailwind CSS**. This project is designed as a simple personal portfolio template that is easy to customize and deploy using GitHub Pages.

---

## 🚀 Live Demo

🔗 Visit the live website here:

**https://codesam04.github.io/portofolio-ku/**

> Replace the URL above with your GitHub Pages link.

---

## ✨ Features

- 📱 **Responsive Design** – Works seamlessly on desktop, tablet, and mobile devices.
- ⚡ **Fast & Lightweight** – Built using pure HTML without heavy JavaScript frameworks.
- 🎨 **Modern UI** – Styled with Tailwind CSS for a clean and elegant appearance.
- 🛠 **Easy to Customize** – All content is contained in a single `index.html` file.
- 🌍 **GitHub Pages Ready** – Can be deployed instantly using GitHub Pages.

## 🛠️ Built With

- HTML5

## 🚀 Deployment

This project can be deployed easily using **GitHub Pages**.

1. Push the project to GitHub.
2. Open **Settings**.
3. Navigate to **Pages**.
4. Select the **main** branch.
5. Save the settings.

Your website will be available at:

```
https://username.github.io/repository-name/
```
